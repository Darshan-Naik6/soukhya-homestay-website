# Soukhya Riverside Homestay Website

A responsive website created for **Soukhya Riverside Homestay** to showcase the property, rooms, and nearby adventure activities.
The website provides visitors with essential information about the homestay and allows them to contact the host for booking enquiries.

## 🌿 Features

* Beautiful homepage introducing the homestay
* Owner / host introduction section
* Room and stay information
* Contact and booking enquiry section
* Information about nearby adventure activities
* Clean and responsive design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 📍 About the Homestay

Soukhya Riverside Homestay offers a peaceful stay surrounded by nature, perfect for travelers looking for relaxation and adventure. Guests can enjoy nearby river activities and explore the natural beauty of the area.
